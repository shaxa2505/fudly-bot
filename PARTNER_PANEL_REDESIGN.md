```

│  │ #103 • Йогурт • 40,000 ₸   │   │

│                                     │
│  📦 Быстрые действия               │
│  [➕ Товар] [📥 Импорт] [⚙️]      │
│                                     │
├─────────────────────────────────────┤
│  [🏠] [📦] [📋] [📊] [⚙️]         │
└─────────────────────────────────────┘
```

### Цветовая схема:
- **Primary**: #FF6B35 (оранжевый - еда)
- **Success**: #10B981 (зелёный)
- **Warning**: #F59E0B (жёлтый)
- **Danger**: #EF4444 (красный)
- **Background**: #F8FAFC (светлый)
- **Card**: #FFFFFF

### Улучшения:
1. **Карточки заказов** - крупные, с аватаром товара
2. **Быстрые действия** - одно нажатие для смены статуса
3. **Реал-тайм обновления** - WebSocket или polling
4. **Свайп действия** - свайп влево = отменить, вправо = подтвердить
5. **Bottom Sheet** для деталей заказа
6. **Haptic feedback** для всех действий
7. **Skeleton loaders** заменить на shimmer cards

## 📱 Новые экраны:

### 1. Dashboard (Главная)
- Статистика за сегодня (карточки)
- Активные заказы (лента)
- Быстрые действия
- Уведомления

### 2. Orders (Заказы)
**Табы:** Новые | Готовятся | Готовы | История

**Карточка заказа:**
```
┌──────────────────────────────┐
│ 🍕 #103                       │
│ Пицца Маргарита × 2          │
│ 40,000 сум                    │
│                               │
│ 👤 Азиз                       │
│ 📞 +998 90 123 45 67         │
│ 📍 Улица Навои, 12           │
│ ⏰ 14:30 - Самовывоз         │
│                               │
│ [✅ Подтвердить] [❌ Отказать]│
└──────────────────────────────┘
```

**Действия:** Свайп вправо = подтвердить, влево = отказать

### 3. Products (Товары)
**Поиск + фильтры:** Все | Активные | Скрытые | Нет в наличии

**Карточка товара:**
```
┌──────────────────────────────┐
│ 📷 [Фото]    Пицца 4 сыра   │
│              45,000 сум      │
│              📦 25 шт        │
│                               │
│ [👁️] [✏️] [📊]               │
└──────────────────────────────┘
```

**Быстрые действия:**
- Тап по количеству = быстрое редактирование
- Toggle для скрыть/показать
- Долгий тап = дополнительное меню

### 4. Stats (Статистика)
**Табы:** Сегодня | Неделя | Месяц

**Графики:**
- Выручка по дням (line chart)
- Заказы по часам (bar chart)
- Топ-5 товаров (horizontal bar)
- Конверсия (pie chart)

### 5. Settings (Настройки)
- Профиль магазина
- График работы
- Зоны доставки
- Уведомления
- Интеграции

# TODO: Интеграция партнёрской статистики

Цель: Подключить кнопку «Статистика сегодня» в режиме партнёра к сервису `app/services/stats.py`.

## План
- Команда/кнопка: найти хендлер кнопки «Статистика» (или добавить) в `handlers/seller/`.
- Получить контекст партнёра: `partner_id`, `store_id` (если применимо), таймзону по профилю/городу.
- Вычислить период «Сегодня»: `start = start_of_day(tz)`, `end = now(tz)`.
- Вызвать: `get_partner_stats(partner_id, Period(start, end, tz), tz, store_id)`.
- Отрендерить карточку: выручка, заказов, товаров продано, активных товаров, средний чек, возвраты (опц.).
- Добавить переключатели периода: Сегодня/Вчера/Неделя/Месяц + кнопку «Обновить» с антифлудом/кэшем.
- Локализация: строки из `locales/` через `localization.py`.

## Технические подсказки
- Кэшировать агрегацию на 30–60 сек (Redis/БД) для частых запросов.
- Учитывать фильтр `store_id` при наличии мультивитрин.
- Логи и ошибки через `logging_config.py`; показывать дружелюбные сообщения.

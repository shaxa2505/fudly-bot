# type: ignore комментарии для подавления ложных Pylance предупреждений

Остальные предупреждения Pylance для динамических атрибутов aiogram являются ожидаемыми и не требуют исправления, так как:

1. `message.from_user` всегда присутствует в реальных сценариях использования Telegram бота
2. `callback.message` всегда доступен для обычных callback query (InaccessibleMessage - редкий edge case)
3. `callback.data` всегда есть для CallbackQuery с data
4. Aiogram использует динамические атрибуты, которые Pylance не может статически проверить

Эти предупреждения не влияют на работу бота и являются особенностью статического анализа для динамического фреймворка.
